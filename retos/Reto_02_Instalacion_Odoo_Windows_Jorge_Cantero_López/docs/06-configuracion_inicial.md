# 06 — Configuración inicial
## Al instalar Odoo y PostgreSQL junto, se han configurado automaticamente muchos de los pasos de la configuracion inicial, sobre todo el puerto 
1. Localiza archivos de **configuración** (si aplica) y el puerto por defecto (ej. `8069`).
2. (Opcional) Ajusta idioma, zona horaria, y otros parámetros iniciales.
3. Anota la **URL de acceso local** (p. ej., `http://localhost:8069`).

- Si se quisiesen hacer estos cambios, tendrias que instalar Odoo y PostgreSQL por separado y en los archivos de Odoo ir a esta carpeta server y abrir el archivo conf y ahí se hacen las configuraciones
